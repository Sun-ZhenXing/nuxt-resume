## 项目经历

### 电商图库系统

- 2022.07 ~ 2022.09，全栈开发
- **项目描述**：给现有电商客户设计和开发图库管理系统，标签化搜索和筛选产品图片，网站服务运行至今
- **主要工作**：
    1. 独立负责产品从 0 到 1 的开发、产品功能迭代
    2. 前端使用 Vue3 + Vite 开发，使用独立设计 UI
    3. 后端使用 Sanic + MySQL，并承担部署和运维
    4. 使用 OpenCV 处理图像，异步工作流完成图像处理
