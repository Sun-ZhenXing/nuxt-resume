# Nuxt 3 Vant 模板项目

[Vant 4 官方文档](https://vant-contrib.gitee.io/vant/#/zh-CN).

## 开始

使用 `corepack` 安装最新版本的 `pnpm`：

```bash
corepack enable
corepack prepare pnpm@latest --activate
```

安装依赖：

```bash
pnpm i
```

开发：

```bash
pnpm dev
```

构建：

```bash
pnpm build
```

静态生成：

```bash
pnpm generate
```
